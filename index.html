Installation

Follow the steps below to setup Admin Panel:

  1. System Requirements:
    • Os: Windows, macOs,Linux
    • RAM: At least 8GB(Android Studio works best with more RAM)
    • Disk Space: At least 10GB of free space
    • A physical phone or a virtual device with Android version 7 or higher is required.
  2. Tools Installation:
    1. JAVA:
      • Windows:
        1. Download OpenJDK 17 LTS
        2. Run the downloaded .msi or .exe installer.
        3. Follow the on-screen instructions. Here, you will find the Java installation path(C:\Program Files\Java\jdk-17) remember this for the Java home setup.
        4. Set JAVA_HOME:
          • Search "Environment Variables" and open it.
          • Under System Variables, find Path and click Edit.
          • Click on 'New' and add the Java installation path, like: (C:\Program Files\Java\jdk-21\bin).
          • Apply & Restart CMD
        5. Verify Java installation by running this command:java -version.
        6. If needed, watch this video for reference: Java Installation Guide .
      • Mac:
        1. Download OpenJDK 17 LTS
        2. Open the downloaded .pkg file.
        3. Follow the on-screen instructions. Here, you will find the Java installation path(/Library/Java/JavaVirtualMachines/temurin-17.jdk/Contents/Home) remember this for the Java home setup.
        4. Set JAVA_HOME:
          • Open the terminal and run this command: 
                                            echo 'export JAVA_HOME=/Library/Java/JavaVirtualMachines/temurin-17.jdk/Contents/Home' >> ~/.zshrc
                                echo 'export CPPFLAGS="-I/opt/homebrew/opt/openjdk@17/include"' >> ~/.zshrc
                                source ~/.zshrc
        5. Verify Java installation by running this command: java -version.
        6. If needed, watch this video for reference: Java Installation Guide .
      • Linux:
        1. Open the terminal and run the following command: 
          sudo apt update && sudo apt install openjdk-17-jdk -y
        2. Verify Java installation by running this command: java -version.
    2. Android Studio:
      • Windows:
        1. Go to the official Android Studio download page: Android Studio Download.
        2. Run the downloaded .exe file.
        3. Follow the on-screen instructions to complete the installation.
        4. Complete the initial setup, including downloading SDK components.
        5. If needed, watch this video for reference: Android Studio Installation Guide .
      • Mac:
        1. Go to the official Android Studio download page: Android Studio Download.
        2. Open the .dmg file and drag Android Studio into the Applications folder.
        3. Open Android Studio from the Applications folder.
        4. Follow the setup wizard to configure Android Studio and download SDK components.
        5. If needed, watch this video for reference: Android Studio Installation Guide .
      • Linux:
        1. Go to the official Android Studio download page: Android Studio Download.
        2. Download the .zip file for your operating system.
        3. Extract the downloaded .zip file.
        4. Open a terminal, navigate to the extracted folder, and run ./studio.sh to launch Android Studio.
        5. Follow the setup wizard to configure Android Studio and download the necessary SDK components.
        6. If needed, watch this video for reference: Android Studio Installation Guide .
    3. Node Js:
      • Windows:
        1. Download and install fnm: winget install Schniz.fnm
        2. Download and install Node.js: fnm install 22
        3. Verify Node Js installation by running this command: node -v .
        4. Verify npm version: npm -v .
      • Mac:
        1. Download and install fnm: curl -o- https://fnm.vercel.app/install | bash
        2. Download and install Node.js: fnm install 22
        3. Verify Node Js installation by running this command: node -v .
        4. Verify npm version: npm -v .
      • Linux:
        1. Download and install fnm: curl -o- https://fnm.vercel.app/install | bash
        2. Download and install Node.js: fnm install 22
        3. Verify Node Js installation by running this command: node -v .
        4. Verify npm version: npm -v .
  3. Visual Studio Code:
    • Windows:
      1. Go to the official Visual Studio Code download page: Visual Studio Code.
      2. Select the Windows version and download the .exe installer.
      3. Run the downloaded .exe file.
      4. Follow the on-screen instructions to complete the installation.
    • Mac:
      1. Go to the official Visual Studio Code page: Visual Studio Code.
      2. Select the macOS version and download the .zip file.
      3. Extract the .zip file and move Visual Studio Code into the Applications folder.

Setup Supabase

Supabase is used as the backend database and authentication provider. Follow these steps to configure it:

  1. Create a Supabase Account:
    • Go to Supabase

    • Click on the 'Start your project' button in Supabase

    • If you already have an account, sign in. Otherwise, click on sign up.

    • Now enter your Gmail, set a strong password, and click on sign up.

    • Check your Gmail and click on the "Confirm Email Address" button.

  2. Create a Supabase Project:

    • After clicking the "Confirm Email Address" button, you will see a page like this.Now click on the "New Project"

    • The 'Create Organization' field in Supabase is optional. You don’t need to make any changes, just click on 'Create Organization'

    • Now, let's create a project. Enter a project name, set a strong password, and select a region. After that, click on the 'Create new project' button.

  3. Adding a User in Supabase Authentication:

    • Finally, you have created the Supabase project. Now, go to 'Authentication', click on 'Add User', and then click on 'Create new User'.

    • Enter the Gmail and password, then click on 'Create User'. Keep in mind, this will be your admin panel username and password.

  4. Upload Flags:
    • Open the link: GitHub - Imran99808/flags
    • Click the green "Code" button.
    • Select "Download ZIP" to download the flags icons,then Extract the ZIP file
    • If the extracted folder has a different name (e.g., flags-main), rename it to flags(ensure the folder name is "flags")

    • Go to Supabase Storage, click New Bucket, set the name as flags (must be exactly "flags"), select Public Bucket, and click Save.

    • Drag and drop the flags folder you downloaded into the Supabase Storage bucket named flags

    • Go to Supabase Storage, navigate to Policies, select the flags bucket, click New Policy, then click "For full customization".

    • Enter a Policy Name, select the appropriate Rights, check the SELECT option, click Review, and then click Save Policy.

  5. Creating Tables and Adding Policies in Supabase:

    • Copy this code, go to Supabase SQL Editor, paste it into the SQL editor, and click the Run button.

      create table "Servers" (
                id uuid primary key default uuid_generate_v4(),
                created_at timestamp default now(),
                country_name text  not null,
                ip text not null,
                flag_url text not null,
                conf text not null,
                port text not null,
                protocol text not null,
                user_name text not null,
                password text not null,
                status boolean not null,
                active boolean not null,
                file_name text not null
                
                );
               alter table "Servers" enable row level security;
              
              
              
               CREATE POLICY "admin read"
               ON public."Servers"
               FOR SELECT
               TO authenticated
               USING (true);
              
              
               CREATE POLICY "admin insert"
               ON public."Servers"
               FOR INSERT
               TO authenticated
               WITH CHECK (true);
              
               CREATE POLICY "admin update"
               ON public."Servers"
               FOR UPDATE
               TO authenticated
               USING (true);
              
              
               CREATE POLICY "admin delete"
               ON public."Servers"
               FOR DELETE
               TO authenticated
               USING (true);
              
               CREATE POLICY "user read"
               ON public."Servers"
               FOR SELECT
               TO anon
               USING ((active = true));
            
            

               create table "Update" (
                      id uuid primary key default uuid_generate_v4(),
                      created_at timestamp default now(),
                      message text unique not null,
                      version text not null,
                     required boolean not null

               );

               INSERT INTO public."Update" (message, version, required) 
               VALUES ('This is a testing update', '1.0', false);


               create table "Users" (
                   id uuid primary key default uuid_generate_v4(),
                   created_at timestamp default now(),
                  fcm_token text unique not null
                  );
  6. Configarations:

    • Now, go to 'Project Settings' and then navigate to 'API'. Here, you will find the URL, public anon, and service secret. Make sure to note them down in a secure place, or keep this page in mind

    • Go to Android Studio, open the strings.xml file (app > res > values > strings.xml), and paste the Supabase URL and public anon key,then save the file (Ctrl + S).

    • Go to Visual Studio Code(CLIENT), open the supabaseConfig.js file (CLIENT > src > supabaseConfig.js), and paste the Supabase URL and service_role secret key, then save the file (Ctrl + S).

Setting Up Package Name in Android Studio

  1. Now we will change the package name. I am using com.own.clientvpn. Your package name should follow the format:

    • com.yourcompanyname.yourappname (must be in lowercase letters)
      • com (or org, net, etc.) is the top-level domain (TLD), usually based on the company's website.
      • yourcompanyname is your unique company or organization name (e.g., own for own.com).
      • yourappname is the specific application name (e.g., clientvpn for a VPN app).

  2. Go to Android Studio, navigate to the app folder, then open the java folder. Select the "com.yourcompanyname.yourappname" package.After that, click on the options(⋮) icon.

  3. Now, select 'Tree Appearance', then uncheck 'Compact Middle Packages'.

  4. Rename the "yourcompanyname" package:

    • Select the "yourcompanyname" package, right-click on it, go to 'Refactor', and click on 'Rename'. After that, click the 'Rename All Directories' button. Replace it with your company name, check both checkboxes, and then click the 'Refactor' button.

    • After that, click on 'Do Refactor'.

  5. Rename the "yourappname" package:
    • Select the "yourappname" package, right-click on it, go to 'Refactor', and click on 'Rename'. After that, click the 'Rename All Directories' button. Replace it with your app name, check both checkboxes, and then click the 'Refactor' button.
    • After that, click on 'Do Refactor'.

  6. Now, click on the menu icon(☰) at the top, then click on 'Sync Project with Gradle Files'.

  7. Go to 'Gradle Scripts', then open build.gradle (Module: app), and copy the package name.

App Customization

  1. Go to Android Studio, then navigate to:app > res > values > strings.xml
  2. Change App Name:
    • Find this line: 
                            <string name="app_name">Y-VpN</string>
    • Replace "Y-VpN" with your app name.
  3. Set Play Store URL with App ID:
    • Find these lines: 
      <!-- Play Store URL -->
      <string name="play_store_url">market://details?id=com.yourcompanyname.yourappname</string>
      <string name="browser_play_store_url">https://play.google.com/store/apps/details?id=com.yourcompanyname.yourappname</string>
    • Replace "com.yourcompanyname.yourappname" with your actual app package name .
  4. Replace with Your Support Email:
    • Find this line: 
      <!--Support email -->
      <string name="feedback_mail">your@gmail.com</string>
    • Replace "your@gmail.com" with your Gmail .
  5. Set Your Privacy Policy Link:
    • Find this line: 
      <!-- privacy-policy link -->
      <string name="privacy_policy_url">https://your-privacy-policy-link.com</string>
    • Replace "https://your-privacy-policy-link.com" with your privacy-policy link .
  6. Update the app title if needed :
    • Find this line: 
      <!-- SPLASH ACTIVITY -->
      <string name="app_title">High Speed Vpn</string>
    • Change "High Speed Vpn" to your desired app title.
  7. Update Terms & Conditions text:
    • Find this line in strings.xml: 
                            <!-- Terms & Conditions Fragment -->
       <string name="conditionText">We will fully adhere to our strict no-logs policy and comply with all relevant data privacy regulations.\n................</string>
    • Replace the placeholder text with your own Terms & Conditions.
  8. Update FAQ questions and answers in strings.xml:
    • Find these lines: 
                            <!-- faq_questions -->
       <string-array name="faq_questions">
      <item> question-1 </item>
      <item> question-2 </item>
      <item> question-3 </item>
      </string-array> <
      !--  faq_answers -->
       <string-array name="faq_answers">
      <item> Answer-1 </item>
      <item> Answer-2 </item>
      <item> Answer-3 </item>
      </string-array>
    • Replace the placeholder text with your own FAQ questions and answers.
    • ⚠️ Ensure the number of <item> elements in faq_questions matches faq_answers (Required).

Admob

Follow the steps below to set up AdMob:

  1. Create an AdMob Account:

    • Go to AdMob
    • Sign in with your Google account.
    • Click on "Get Started" and follow the setup process.

  2. Add a New App to AdMob:

    • Go to "Apps" and click the "Add your first app" button.
    • Select Platform: Choose Android and Select whether your app is listed on the Play Store or not.
    • Click the Continue button
    • Enter App Name
    • Click "Add"
    • go to Apps, select your app, then navigate to App Settings. Copy the App ID, then open Android Studio > app/res/values/strings.xml, find <string name="admob_app_id">ca-app-pub-3940.......</string>, and replace the test App ID with your own.

  3. Create an Ad Unit:

    • Go to "Ad Units" → Click "Get Started"
    • Select an "Interstitial" Ad format
    • Enter Ad Unit Name
    • Click "Create" and Copy the Ad Unit ID, then open Android Studio > app/res/values/strings.xml, find <string name="admob_unit_id">ca-app-pub-3940.......</string>, and replace the test unit ID with your own.
